# Streamlit приложение

Интерактивное веб-приложение для демонстрации модели предсказания цен на квартиры в Москве.

## Возможности

- Ввод параметров квартиры (площадь, количество комнат, этаж и т.д.)
- Предсказание цены на основе обученной модели
- Визуализация факторов, влияющих на цену
- Интерактивные графики и статистика

## Развертывание

### Локально

```bash
source ../venv/bin/activate
pip install streamlit
streamlit run app.py
```

### Hugging Face Spaces

1. Создайте новый Space на [Hugging Face](https://huggingface.co/spaces)
2. Выберите SDK: Streamlit
3. Загрузите файлы из этой директории
4. Приложение будет автоматически развернуто

## Требования

- Python 3.8+
- streamlit
- pandas
- numpy
- scikit-learn (или другая ML библиотека)
- plotly (для визуализаций)
